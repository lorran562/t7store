# T7 Store ⚽

Loja de camisas de futebol — Next.js 14 + Tailwind CSS + TypeScript.

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Deploy:** Vercel

## Estrutura

```
app/          → layout e página principal
components/   → Header, Hero, Ticker, ProductCard, ProductModal, CartDrawer, Footer, WhatsAppFloat
context/      → CartContext, ToastContext
lib/          → dados dos produtos e utilitários
public/       → logo e assets estáticos
```
